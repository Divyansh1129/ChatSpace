# ChatSpace

ChatSpace is a full-stack chat application where users can send messages and share images in real-time.

---

## Features

* User authentication (Login / Signup)
* Real-time chat
* Image sharing
* Responsive UI

---

## Tech Stack

* Frontend: React, Tailwind CSS
* Backend: Node.js, Express
* Database: MongoDB
* Other: JWT, Cloudinary

---

## Setup

### Clone repository

```
git clone https://github.com/Divyansh1129/ChatSpace.git
cd ChatSpace
```

### Backend

```
cd Backend
npm install
npm start
```

### Frontend

```
cd Frontend
npm install
npm run dev
```

---

## Environment Variables

Create a `.env` file in Backend:

```
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
CLOUDINARY_CLOUD_NAME=your_name
CLOUDINARY_API_KEY=your_key
CLOUDINARY_API_SECRET=your_secret
```
